Genesys Cloud Widget

Place a call

Upload-ready single-page widget that authenticates the signed-in user with OAuth PKCE and posts to /api/v2/conversations/calls when Place Call is selected. The JSON body uses Dialed Number as phoneNumber and My Number as callerId.

OAuth Configuration

Redirect URI:
Auth status: Signed out

Configure your OAuth client in Genesys Cloud for the PKCE grant and add this page URL as an authorized redirect URI. This page stores the client ID and token only in the browser session.
